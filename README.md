# ecommerce-sql-joins
SQL joins analysis on ecommerce dataset

 Ecommerce SQL Joins Project

## Description
This project demonstrates SQL join operations on an eCommerce dataset including customers, orders, products, and categories.

## Files Included
- joins_queries.sql : SQL queries with INNER JOIN, LEFT JOIN, and aggregation
- joined_output.csv : Exported join results
- insights.txt : Business insights derived from data analysis

## Key Skills
- SQL JOINs
- Data Analysis
- Business Insights
- MySQL
